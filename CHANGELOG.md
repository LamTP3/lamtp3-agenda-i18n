# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0-alpha.2] - 2026-01-27

### Added

- Added `i18n` property to configuration. This allows users to pass a custom dictionary object to override specific strings or provide a full custom language pack.

### Changed

- Changed default language from `'vi'` (Vietnamese) to `'en'` (English).

## [1.0.0-alpha.1] - 2026-01-20

### Added

- Fork from Agenda v5.0.0.
- Started internationalization (i18n) support.
- Initial preparation for multi-language job definitions and metadata.
